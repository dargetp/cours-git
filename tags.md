# Les tags

Les **tags** indiquent une version du projet.
Ils sont liés à un **commit**.

Ajouter un tag:
```sh
git tag v1.0.0
```

Sementic versioning:
https://semver.org/

Changelog:
https://keepachangelog.com/en/1.0.0/

Examples:
https://github.com/AymDev/Fregata/releases
https://github.com/AymDev/Fregata/blob/master/CHANGELOG.md
